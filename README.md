# Sumit Kumar - Beautiful Jekyll Portfolio (Custom)

This repository is a customized Beautiful Jekyll template for Sumit Kumar.
Upload to your `ksumit-webdeveloper.github.io` repository (or replace its contents) and enable GitHub Pages (main branch, root) to publish your site.

Files included:
- _config.yml (configured with your details)
- index.md (single-page with About, Websites I've Developed, Pricing, Contact)
- blog.md (archive layout)
- _posts/ (two sample posts)
- assets/img/favicon.png (your SK favicon)
